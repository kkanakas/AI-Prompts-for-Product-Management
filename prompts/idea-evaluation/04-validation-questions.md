# Validation Interview Questions

**Phase:** Idea Evaluation
**Purpose:** Generate open-ended questions to validate problem-solution fit with real users.

## Prompt Template

```
Feature idea: [your idea and the problem it solves]

Generate questions to ask users that would:
1. Validate they have this problem
2. Test if our solution approach makes sense (problem-solution fit)
3. Identify if they would actually use this vs. current workaround
4. Surface what is missing or what would make them NOT use it
5. Understand willingness to pay or prioritization

Provide 5 to 7 interview questions (open-ended, avoid leading).
```

## Placeholders

| Variable | Description |
|---|---|
| `[your idea and the problem it solves]` | The feature concept and user need |

## Tips

- Avoid leading questions ("Would you love a feature that...?")
- Focus on past behavior, not hypothetical futures
- Combine with the Assumption Mapping prompt (03) to target your riskiest bets
