# User Journey Map Generation

**Phase:** Design and Product Requirements
**Purpose:**: Generate a user journey map documentation for a Feature

## Prompt Template

```

The product feature is  [FEATURE DESCRIPTION]. The flow of activies [GENERAL FLOW OF ACTIVITY OR ACTIVITIES]

OUTPUT: 
1. Purpose and Scope
2. Overview of the document 
3. Sequence diagram of interactions [MERMAID SEQUENCE DIAGRAM]
4. For each stage of the activities
             - Activity Description 
             - User Goals 
             - Customer Experience
             - System context 

5. Key takeaway

```

## Placeholders

| Variable | Description |

| --- | --- |

|`[FEATURE DESCRIPTION]`| Description of the Feature |
| `[GENERAL FLOW OF ACTIVITY OR ACTIVITIES]` | link to documents or text descrition of the flow of activities |
| `[MERMAID SEQUENCE DIAGRAM]` |  Generate a Mermaid sequence diagram and embeded it in the document |
