# VRIO Analysis

**Phase:** Ideation
**Purpose:** Evaluate whether a product capability, feature, or asset provides a sustainable competitive advantage using the VRIO framework (Valuable, Rare, Inimitable, Organized).

## Prompt Template

```
I am a product manager evaluating the following capability or asset:

Capability / Asset: [CAPABILITY_OR_ASSET]
Product context: [PRODUCT_CONTEXT]
Target market: [TARGET_MARKET]
Key competitors: [KEY_COMPETITORS]

Run a VRIO analysis on this capability to assess whether it represents a source of sustainable competitive advantage. For each dimension, provide:

1. Assessment — does this capability meet the criterion? (Yes / Partially / No)
2. Reasoning — explain why, with specific reference to our product and market
3. Evidence or assumptions — what data or signals support this assessment
4. Implication — what does this mean for our product strategy?

VRIO dimensions to cover:
- V — Valuable: Does this capability allow us to exploit an opportunity or neutralize a competitive threat? Does it create meaningful value for users?
- R — Rare: Is this capability currently possessed by few or none of our competitors?
- I — Inimitable: Would it be difficult, costly, or time-consuming for competitors to copy or substitute this capability? Consider path dependency, causal ambiguity, and social complexity.
- O — Organized: Is our organization structured to actually capture the value from this capability? Consider teams, processes, incentives, and tooling.

After completing all four dimensions, provide:
- An overall competitive position rating: Competitive Disadvantage / Competitive Parity / Temporary Advantage / Sustained Advantage
- A strategic recommendation: whether to invest, protect, divest, or build organizational support around this capability
- One key risk that could erode this advantage over time
```

## Placeholders

| Variable | Description |
|---|---|
| `[CAPABILITY_OR_ASSET]` | The specific feature, technology, dataset, network effect, brand, or process being evaluated |
| `[PRODUCT_CONTEXT]` | Brief description of the product, its stage, and how this capability fits into it |
| `[TARGET_MARKET]` | The market segment or buyer persona this capability is meant to serve |
| `[KEY_COMPETITORS]` | Direct and indirect competitors relevant to this capability |

## Tips

- VRIO is most useful when evaluating whether to double down on a differentiator or deprioritize something that feels unique but isn't strategically defensible.
- The "Organized" dimension is often underweighted — a valuable, rare, and hard-to-copy capability still fails if the team, incentives, or processes aren't set up to deliver it.
- Run this analysis on 2–3 competing capabilities side by side to surface where your real moat lies versus where you only have temporary parity.
- Revisit VRIO analyses quarterly — competitive landscapes shift and what was inimitable last year may be table stakes today.
