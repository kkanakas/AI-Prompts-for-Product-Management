# Jobs-to-be-Done Analysis

**Phase:** Customer Discovery
**Purpose:** Analyze interviews through the JTBD lens — functional, emotional, and social jobs.

## Prompt Template

```
Analyze these interviews through Jobs-to-be-Done lens:

For [PRODUCT/SOLUTION users currently use]:
1. What "job" are users hiring it to do?
   - Functional job
   - Emotional job
   - Social job
2. What triggers them to seek this solution?
3. Where does current solution succeed at the job?
4. Where does it fail or create friction?
5. What workarounds have users created?
6. What would make them "fire" current solution and "hire" ours?

Use actual user language. Provide quotes supporting each finding.
```

## Placeholders

| Variable | Description |
|---|---|
| `[PRODUCT/SOLUTION users currently use]` | The existing tool or process users rely on today |
