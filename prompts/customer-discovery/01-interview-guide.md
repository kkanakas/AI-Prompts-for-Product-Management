# Interview Guide Generator

**Phase:** Customer Discovery
**Purpose:** Create comprehensive interview guides with probing questions and edge case coverage.

## Prompt Template

```
I am researching [PROBLEM] for [USER TYPE]. I want to understand:
- Current workflows
- Motivations, Needs, and Pain-points
- Decision criteria and willingness to pay for a solution
I also would like to identify patterns and edge cases. What frustrations do people have in the current environment? What is the impact of the problem for them?

Create an interview guide with:
1. Main questions exploring each area
2. Follow-up probes for each question
3. Include "why" and "tell me of a time when" questions
4. Questions that might be useful for edge cases
```

## Placeholders

| Variable | Description | Example |
|---|---|---|
| `[PROBLEM]` | The problem you're investigating | "time tracking and invoicing for freelancers" |
| `[USER TYPE]` | Who you're researching | "freelance designers earning $50-150K/year" |

## Tips

- Be as specific as possible about the problem space
- Include any known constraints (industry, geography, company size)
- If you already have some interview data, mention what you've learned so far
