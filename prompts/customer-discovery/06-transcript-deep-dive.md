# Theme Deep Dive

**Phase:** Customer Discovery
**Purpose:** Drill into a specific theme identified in transcript analysis.

## Prompt Template

```
You identified "[THEME]" as a recurring pattern in my interviews.

Let's go deeper:
1. Show me every instance where this came up (quote + speaker)
2. Is this truly one theme or multiple related issues?
3. What's the root cause? (look for why this is a problem)
4. Who experiences this most acutely? (role, context, segment)
5. How are users currently solving/working around this?
6. What would "success" look like for solving this?

Then: What 3-5 questions should I ask in follow-up interviews
to validate and deepen understanding of this theme?
```

## Placeholders

| Variable | Description |
|---|---|
| `[THEME]` | The theme name from your initial analysis |
